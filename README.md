# YouTube Intelligence Engine

An AI-powered NLP system that scrapes and analyzes YouTube comments to generate structured insights using sentiment analysis, keyword extraction, named entity recognition (NER), topic modeling, and Retrieval-Augmented Generation (RAG).

## Project Overview

This project transforms large-scale YouTube discussions into searchable and analyzable knowledge. The system retrieves YouTube videos related to a selected topic, scrapes comments using the YouTube Data API, preprocesses noisy text data, and applies multiple NLP techniques for analysis and question answering.
